# **App Name**: ima | Sí Vale Landing Page

## Core Features:

- Hero Section: Display a prominent hero section with a headline, description, and a call-to-action button.
- "¿Qué es ima?" Section: Introduce the "ima" concept with a brief description and feature highlights.
- Feature Cards: Showcase key benefits with individual cards highlighting "Consultas ilimitadas", "Nutrición personalizada", and "Soporte psicológico".
- "Así de fácil es activar tu beneficio" Section: Guide users through the activation process in 3 simple steps.
- Final Call-to-Action Section: Prompt users to sign up with a final message and a prominent call-to-action button.
- Footer: Include a standard footer with links to Legal, Ayuda, and Sitios sections.

## Style Guidelines:

- Background color: Dark teal (#042326) for a professional and modern look.
- Text and Accent color: Lime green (#D2F252) for contrast and highlights.
- Body and headline font: 'Sora' (sans-serif) for a modern and readable aesthetic. Note: currently only Google Fonts are supported.
- Font for "ima": 'BD Colonius Regular' Note: currently only Google Fonts are supported.
- Clean, minimal layout with clear sections to guide the user's eye.
- Use of simple, consistent icons to represent the features.
